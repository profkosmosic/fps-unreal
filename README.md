Drone Destroyer is a small First Person Shooter game built with Unreal Engine 5 and Blueprints where you must eliminate rogue drones that have overtaken a military base before time runs out.

# CONTROLS
- WASD to Move
- Mouse to Look
- Left Click to Shoot
- Right Click to Aim Down Sights
- Space to Jump
